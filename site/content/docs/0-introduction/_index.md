# 導讀

生時間（Make Time）不是要談生產力，而是幫你於日常創造更多時間來做你真正在乎的事，讓生活歸你所有。21 世紀有兩股強大力量在爭搶你的每分每秒：馬不停蹄（工作一刻都不該得閒），與萬丈深淵（永遠在更新的娛樂）。它們已成為生活的一種預設值，提高生產力不會是解方，用意志抵抗也不會是出路，生時間則可選擇你想聚焦的事、提振做它的活力、打破預設值循環，憑己願選擇生活方式。

這架構衍生於執行設計衝刺（Design Sprint）所發現的事實，它教會作者們四件事：

- 用某重要目標開啟一天時，神奇的事會發生：對此目標極具企圖心，但只關注一件事
- 禁用裝置時，能做更多事：人們會全神貫注手邊工作
- 專心工作和清晰的思考非常重要：健康食物、散步運動、頻繁休息都有幫助
- 實驗能改善過程：親眼見證改變成果，會給予人們更多信心
