# 雷射

和 email 搞外遇
重新設計 YouTube
為什麼「萬丈深淵」那麼難以抗拒
別等科技還你時間
創造分心的障礙

像雷射般專注於精華，是很棒的感覺。然而我們也常會因各種事務分心，尤其是萬丈深淵的誘惑。它們容易上癮的理由有四：人們對科技（發明新事物）的熱情、科技產品會進化、產品隨時會有競爭、和人們仍像穴居時期－易分心，喜愛不可預測的報酬。

要擊敗分心，就得讓它難反應：創造點不便，更易把活力導向生時間。當我們貫注於雷射模式，就能為最重要的事生出高品質的時間。改變自身優先順序，展現對時間的計較，也能提醒他人從萬丈深淵中抽身。
