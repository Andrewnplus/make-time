# 精華

遺忘的時光
你今天的「精華」是什麼？
挑選「精華」的三種方式
信任直覺，選擇最好的「精華」

精華不是唯一要做的事，而是所給予的焦點：我們藉選擇要關注的來創造自身現實。這可用三個標準來選擇：

1. 急迫性：從待辦清單、Email、行事曆上尋找有時效性、重要、中等規模的案子
2. 滿足感：著眼於你想完成的－那些不急迫，想做但尚未花時間的計畫
3. 樂趣：對它們投入其實不浪費時間，對運用時間毫無頭緒才是種浪費

最好的選擇法是信任直覺－找出急迫、愉快、滿足的精華；選擇的經驗是選那些會佔 1－1.5 小時的精華－過短很難聚精會神，過長則會需要喘口氣。
